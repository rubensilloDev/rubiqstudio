# Instrucciones de Diseño y Estructura por Sección

Este archivo sirve como guía obligatoria para cualquier IA o desarrollador que trabaje en el diseño e implementación de las distintas secciones del proyecto. Cada sección tiene su estructura, elementos y colores definidos en un archivo Markdown específico dentro de la carpeta \skills\. Debe seguirse estrictamente cada especificación.

## Índice de Secciones y Archivos de Referencia

1. **Sección 1: Hero Principal (Desarrollo web a medida)**
   - **Descripción:** Layout de dos columnas con fondo degradado y efecto de destello, textos principales y botón CTA.
   - **Ruta del archivo:** [skills/hero.md](./skills/hero.md)

2. **Sección 2: Llamada a la Acción Central (Estrategia)**
   - **Descripción:** Sección de una sola columna centralizada con título en relieve 3D y botón CTA blanco.
   - **Ruta del archivo:** [skills/seccion1.md](./skills/seccion1.md)

3. **Sección 3: Tarjetas de Precios**
   - **Descripción:** Título superior y dos columnas de tarjetas (Landing Page y Sitio Web Corporativo) con fila informativa inferior.
   - **Ruta del archivo:** [skills/seccion2.md](./skills/seccion2.md)

4. **Sección 4: Llamada a la Acción (Rediseño)**
   - **Descripción:** Layout de dos columnas (70% texto y 30% CTA) con tipografía destacada.
   - **Ruta del archivo:** [skills/seccion3.md](./skills/seccion3.md)

## Reglas de Implementación
- **Cumplimiento estricto:** No se debe modificar la estructura, clases base ni los esquemas de color definidos en cada archivo de skill.
- **Estilos globales:** Usar las variables declaradas en el \:root\ de \global.css\ según se indica en cada especificación.
